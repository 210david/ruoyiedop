# RuoYi-Vue3 项目目录结构分析

> 本项目基于 **Spring Boot 4.0.6 + JDK 17 + MyBatis + Vue 3 + Element Plus + Vite + Pinia** 的企业级管理系统，采用 **Maven 多模块** 架构，包含一个自研 **WMS 仓库管理系统**。

---

## 一、顶层结构总览

```
RuoYi-Vue3/
├── pom.xml                  ← Maven 父级 POM，管理 7 个子模块
├── ruoyi-admin/             ← 启动模块（Web 入口）
├── ruoyi-framework/         ← 框架核心模块（安全/数据源/AOP/拦截器）
├── ruoyi-system/            ← 系统管理模块（用户/角色/菜单/部门/字典等）
├── ruoyi-common/            ← 通用工具模块（常量/异常/工具类/注解）
├── ruoyi-quartz/            ← 定时任务模块
├── ruoyi-generator/         ← 代码生成器模块
├── ruoyi-wms/               ← WMS 仓库管理模块（自研业务）
├── ruoyi-ui/                ← 前端工程（Vue 3 + Vite）
├── sql/                     ← 数据库脚本
├── bin/                     ← 运维脚本
├── doc/                     ← 文档
├── docker-compose.yml       ← Docker 编排
├── Dockerfile               ← Docker 构建文件
├── ry.bat / ry.sh           ← 启动脚本
└── logs/                    ← 日志目录
```

## 二、各模块详细分析

### 1. ruoyi-admin — 启动入口模块

**职责**：整个系统的启动入口，包含 `main` 方法，承载所有 Controller（API 控制器）。

```
ruoyi-admin/src/main/java/com/ruoyi/
├── RuoYiApplication.java          ← Spring Boot 主启动类
├── RuoYiServletInitializer.java   ← 外部容器部署初始化
└── web/
    ├── controller/
    │   ├── common/    ← 公共接口：验证码(Captcha)、文件上传下载(Common)
    │   ├── monitor/   ← 监控接口：服务器监控、缓存监控、登录日志、操作日志、在线用户、健康检查、应用监控
    │   ├── system/    ← 系统管理接口：用户/角色/菜单/部门/岗位/字典/配置/通知/登录/注册/个人中心/配置备份/撤销日志
    │   └── tool/      ← 测试工具接口
    └── core/config/
        └── SwaggerConfig.java   ← OpenAPI/Swagger 文档配置

src/main/resources/
├── application.yml          ← 主配置文件（端口 8081、Redis、MyBatis 等）
├── application-druid.yml    ← Druid 数据源配置
├── logback.xml              ← 日志配置
├── banner.txt               ← 启动横幅
├── i18n/                    ← 国际化资源文件
├── mybatis/                 ← MyBatis 全局配置
└── META-INF/                ← Spring Boot 自动装配配置
```

**依赖关系**：依赖 `ruoyi-framework`、`ruoyi-quartz`、`ruoyi-generator`、`ruoyi-wms`

## 三、模块依赖关系图

```
                    ┌──────────────┐
                    │  ruoyi-admin  │ ← 启动入口
                    └──────┬───────┘
           ┌───────────────┼───────────────┐
           ▼               ▼               ▼
    ┌──────────────┐ ┌───────────┐ ┌───────────┐
    │ruoyi-framework│ │ruoyi-quartz│ │ruoyi-generator│
    └──────┬───────┘ └─────┬─────┘ └─────┬─────┘
           │                │              │
           ▼                ▼              ▼
    ┌──────────────┐  ┌───────────────────────┐
    │ ruoyi-system  │  │     ruoyi-common       │ ← 最底层，无依赖
    └──────┬───────┘  └───────────────────────┘
           │
           ▼
    ┌──────────────┐
    │  ruoyi-wms   │ ← 依赖 system + common
    └──────────────┘
```

**依赖说明**：

- `ruoyi-common`：最底层模块，不依赖任何其他模块，被所有模块引用
- `ruoyi-system`：依赖 `ruoyi-common`，提供系统管理业务
- `ruoyi-framework`：依赖 `ruoyi-system` + `ruoyi-common`，提供框架基础设施
- `ruoyi-wms`：依赖 `ruoyi-system` + `ruoyi-common`，提供仓库管理业务
- `ruoyi-quartz`：依赖 `ruoyi-common`，提供定时任务
- `ruoyi-generator`：依赖 `ruoyi-common`，提供代码生成
- `ruoyi-admin`：依赖以上所有模块，作为启动入口聚合所有功能

## 四、技术栈总结

| 层面 | 技术选型 |
|------|---------|
| **后端框架** | Spring Boot 4.0.6, JDK 17 |
| **ORM** | MyBatis (原生) + PageHelper 分页 |
| **数据库** | MySQL 8.0 + Druid 连接池 |
| **缓存** | Redis (FastJson2 序列化) |
| **安全** | Spring Security + JWT |
| **定时任务** | Quartz |
| **代码生成** | Velocity 模板引擎 |
| **前端框架** | Vue 3.5 + Element Plus |
| **构建工具** | Vite |
| **状态管理** | Pinia |
| **HTTP 客户端** | Axios |
| **API 文档** | SpringDoc OpenAPI (Swagger) |
